ウェルズの活躍

1930年代、ウェルズは書籍という線形な型には収まらない情報を蓄積する ” World Brain ” の創設を提案している
彼は「タイムマシン」などの小説で名声を得たのち、戦争を根絶するために国際連盟の提唱などの社会活動をはじめた
百科全書運動 ” New Encyclopaedism ” もそのひとつで、世界を知的に統合し、世界平和の基盤とすることを目的としている
ウェルズの世界百科事典 ” World Encyclopaedia ” の構想は、インターネットやウィキペディアの原型ともいえる、クロスリファレンスの提唱だった
情報の関連性を辿り、大量の文献を逐一調べることなく目的の情報に到達する事を、ハイパーテキストは可能にした

WWW

1990年12月20日、ティモシー・ジョン・バーナーズ＝リーは、世界最初のウェブサイトを公開した
世界中に張り巡らした文書間のつながりが、蜘蛛の巣を連想させることから、それは ” World Wide Web ” と名付けられた
記述には主にHTMLやXHTMLといったハイパーテキストのマークアップ言語が使用され、インターネット上に散在する文書同士を相互に参照可能にする
閲覧者はハイパーリンクが付された箇所をクリック、タップする事で、先の文書を表示させることができる
ウェブは、人類史上最大の規模で情報交換を可能にした
情報提供を担う者はウェブサーバを公開し、一般利用者は、ウェブブラウザを介して、ウェブサーバにある情報を閲覧するシステムだ
サーバ内に存在するHTMLドキュメントの各種情報を、インターネットプロトコルスイート ” Transmission Control Protocol / Internet Protocol ” を介して、アプリケーションプログラミングインタフェースに送信する
物理層、データリンク層、ベンダ、OSが異なっていても、このプロトコルに対応している機器同士であれば、相互に通信することが可能となる

血

赤ちゃんの赤は血の色
分娩のときに陣痛の圧力で胎盤の血液が新生児の体内に絞り出されると、いささか多血症になり皮膚の色が赤く見える
これが赤ちゃんの語源で、生まれたばかりの子供は、新芽や若葉のように生命力溢れていることから、みどりごとも呼ぶ
血液は心臓によって加圧され、動脈を通じて全身へ送られる
毛細血管に達すると細胞間質液に栄養分、酸素などを放出し、静脈を経て心臓へと戻る
赤い血は、 ヘモグロビンの鉄と酸素が結びついて、さびた色
エビやザリガニなどの節足動物 、タコやイカなどの軟体動物の血は、ヘモシアニンという物質が酸素を運ぶ
ヘモシアニンには、鉄の代わりに銅がたくさん含まれているので、血は銅がさびた色､つまり､古い銅像のような緑から、薄い青色になる
血は生命の象徴
血は、血統、血脈、血族、血のつながりや血縁というように、親族関係、遺伝に結びつけられて語られる
動物、菌類、植物、原生生物、古細菌、細菌などの生物は、すべて細胞によって構成される
ウイルスも、寄生する細胞がなくては増殖できない
生物の主要な構成成分は、水、タンパク質、脂質、多糖、核酸からなり、体の約70％は水、その他の物質が30％ほどを占める
血の流れは、ペストやマラリアなどが伝染する経路ともなる
14世紀に流行したペストによって、当時のヨーロッパ人口の3分の1が死亡した
また、他人の血液が付着した針で指を刺したり、自分の体に付着した場合は、B型肝炎ウイルス、C型肝炎ウイルス、ヒト免疫不全ウイルスの感染を考えなければならない
1990年にカールウーズが提唱した3ドメイン説では、地球上の全ての生物は、生物のもつタンパク質のアミノ酸配列や遺伝子の塩基配列を用いて系統解析され、細菌、古細菌、真核生物のいずれかに分類される
植物と動物は細胞レベルで見ると、DNA複製のメカニズムや、細胞膜の主成分などは共通性が高い
RNAの核酸塩基は、アデニン、グアニン、シトシン、ウラシルの4種で構成されている
アデニン、グアニン、シトシンはDNAにも同じ構造が見られるが、DNAにおけるチミンが、RNAではウラシルに置き換わり、相補的な塩基はアデニンとなる
チミンとウラシルは非常に似た塩基で、共にピリミジン環を持っている
DNAとRNAは、ともにヌクレオチドの重合体である核酸だけど、両者の生体内の役割は異なっている
DNAは主に、核の中での情報の蓄積、保存を担う
いっぽうRNAは、その情報の一時的な処理を担い、DNAと比べて頻繁に合成、分解されてしまう
RNAはDNAを鋳型として、転写という作用によって合成されるのだ

バイオハザード

ウイルスや細菌、リケッチア、寄生虫、真菌などの病毒で、人畜に感染し伝播によって害をなす生物災害
1876年、ロベルトコッホが炭疽菌の純粋培養に成功して以降、注射針やピペットを介してチフス菌、ブルセラ菌、破傷風菌、コレラ菌、ジフテリア菌との実験室感染が相次いで起こってしまう
こういった惨事を契機にバイオセーフティーの必要性が認知されるようになった
肝炎ウイルスや結核菌、エキノコックス、プリオンタンパク質といった病原体の培養物やその廃棄物、注射針等の医療廃棄物は、とくに病毒をうつしやすい
炭疽菌、結核菌、コレラ菌の発見者であるコッホは、感染症の病原体を特定するための指針、コッホの原則を提唱し医学の発展に貢献した
管理等級を決め、より危険なバイオハザード物質を扱う部屋ほど多くの対策が必要となる
梅毒の末期症状である、麻痺性痴呆と脊髄癆
研究者たちは、梅毒患者の脳や脊髄から、梅毒スピロヘータを見つけ出そうと試みていたが、上手くいかなかった
数千枚にもおよぶ病理組織標本の観察によって、菌がいると考えられていた血管まわりではなく、繊維の奥深くでスピロヘータは確認された
細菌の形態には、コレラ菌などの球菌、スピリルム、カンピロバクター、ヘリコバクターといった桿菌の他に、細長い菌体がらせん形になったものが存在し、微生物の体系的な研究をしたエーレンベルグはこれをスピロヘータと名付けた
グラム陰性の常在菌で、一部はヒトに対して、梅毒、回帰熱、ライム病などの病原性を持つ
最初に微生物を発見したことで知られるレーウェンフックが、1683年9月、イギリス王立協会に送ったスケッチにも描かれている
梅毒はスピロヘータという種類に分類される真正細菌のうち、トレポネーマと呼ばれるもののひとつが引き起こすのが定番だそう
暗視野顕微鏡で観察され、1998 年に全ゲノムのDNA配列が決定、公開されたが、培養の困難さなどから現在でも病原性の機構はほとんど解明されていない
ヒトが唯一の自然宿主で低酸素状態でしか長く生存できない
感染経路は限定され、感染者の皮膚および粘膜からの滲出液、唾液、精液、血液、膣分泌液などが感染源となる
症状が出ていない間も体内は悪化の一途を辿っているので、患者は完治診断を受けるまでペニシリン系のアモキシシリン投与を受ける

ヴェーダ

人がこの世で行った業、カルマが原因となって、次の輪廻の生まれ変わりが決まる
人は悲惨な状態に生まれ変わる事に不安を抱き、六道輪廻の運命から抜け出す解脱の道を求める
業の教えの始まりは、ヴェーダの付属文献の一つ「ウパニシャッド」に見られ、インドのカースト制度を支える思想になった
ウパニシャッドにおいてオウム ॐ は、宇宙の根本原理であり、バラモン教の聖職者を指す言葉でもあるブラフマンを象徴するものとされる
ウパニシャッドはバラモンの師から弟子に伝承された奥義を意味し、その文献は、紀元前500年頃までに編纂されたと言われている

ヴェーダの神々は元々、征服民族アーリヤ人の伝承だった
彼らはインドを征服する過程で、身分制度「ヴァルナ制」を形成した
「アーリヤ」とは、高貴な人を意味し、彼らは先住民を奴隷「ダーサ」として区別したのだ
制度はインド社会に深く根を下ろし、グプタ朝時代には、ヒンドゥー教として一つの宗教になる ヒンドゥー教はバラモン教をベースに、インドの民間信仰や習俗を融合して作られた
聖典として「ヴェーダ」があるほか「マハーバーラタ」「ラーマーヤナ」「プラーナ」といった叙事詩が重要視されている

紀元前2300～紀元前1800年の間、インド亜大陸は青銅器文明が栄えた
その名残として今でも、インダス川上流、パンジャーブ地方には「ハラッパー遺跡」が、インダス川下流のシンド地方には「モヘンジョダロ遺跡」が残っている
この文明の滅亡原因に、侵入民族であるアーリア人の破壊説もあったが、年代のズレがあり、この説は否定されている

叙事詩には、インドラの雷によって焼きつくされ滅亡したのは、モヘンジョダロだと書かれている
インドラとは日本で帝釈天と呼ばれている雷神で、そのルーツはインド＝イラン共通時代までさかのぼる
紀元前14世紀に、ヒッタイトとミタンニとの間で結ばれた条文の中に名前があり、小アジアやメソポタミアなどでも信仰されていたらしい
